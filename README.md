# Camper Cafe Menu

This is a simple responsive cafe menu webpage showcasing coffee and dessert items with prices. It uses HTML and CSS to present the menu in an attractive, clean layout with background imagery and styled typography.

---

## Features

- Displays a cafe name with establishment year.
- Two main sections: Coffee and Desserts, each with images and menu items.
- Items list with flavor/dessert names aligned left and prices aligned right.
- Responsive center-aligned layout with a warm, cafe-inspired color scheme.
- Footer with a website link and cafe address.
- Background image of coffee beans for ambiance.

---

## Project Structure

/project-folder
├── index.html # Main HTML file
├── style.css # CSS file with styling rules
└── images/
├── beans.jpg # Background image
├── coffee.jpg # Coffee section image
└── pie.jpg # Dessert section image

---

## How to Run

1. Download or clone the project folder.
2. Ensure the `images` folder contains the required image files.
3. Open `index.html` in any modern web browser.
4. View the cafe menu with coffee and dessert items styled with background image and formatting.

---

## Customization

- Update the menu items and prices in the HTML `<article class="item">` sections.
- Replace images in the `images` folder to customize appearance.
- Modify `style.css` to change colors, fonts, spacing, or layout styles.
- Adjust font sizes and container width for different screen sizes if needed.
